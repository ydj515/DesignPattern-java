# DesignPattern-java


## 디자인 패턴 종류


### 1. Creational Pattern
#### - 객체 생성에 관련된 패턴
* Singleton
* Factory Method
* Abstract Factory


### 2. Structual Pattern
#### - 클래스나 객체를 조합해 더 큰구조를 만드는 패턴
* Composite
* Decorator

### 3. Behavioral Pattern
#### - 클래스 사이의 알고리즘이나 책임 분배에 관련된 패턴
* Command
* Observer
* State
* Strategy
* Template Method
